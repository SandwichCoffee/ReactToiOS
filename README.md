# ReactToiOS

React + Spring Boot 기반 기존 관리자 서비스를 Swift iOS 앱으로 포팅하는 프로젝트입니다.

## Goal
- 기존 백엔드를 재사용해 iOS 클라이언트 출시
- MVP 기능을 빠르게 완성하고 TestFlight 배포

## Scope (MVP)
- 로그인/회원가입
- 상품 목록/상세
- 장바구니 조회/추가/수정/삭제
- 주문 생성
- 내 계정 기본 화면

## Tech (planned)
- SwiftUI
- MVVM
- async/await + URLSession
- Keychain (토큰 저장)
