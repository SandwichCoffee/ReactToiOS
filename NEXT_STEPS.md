# NEXT STEPS

## Immediate
1. Xcode에서 `ReactToiOS` SwiftUI App 프로젝트 생성
2. 번들 ID 결정 (`com.<yourname>.reacttoios`)
3. 최소 지원 버전 설정 (iOS 17+ 권장)
4. `AuthService`, `ProductService`, `CartService` 네트워크 계층 생성

## First Milestone
- 로그인 성공 후 상품 목록 진입까지 구현
- 토큰 Keychain 저장/복원 동작 확인

## Second Milestone
- 장바구니/주문 플로우 완성
- 공통 에러 처리(401/403/409/500) 화면 반영

## Release Milestone
- TestFlight 내부 배포
- 스크린샷/앱 설명/개인정보 처리방침 정리
